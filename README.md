# Billiard Practice Tracker

## 1. Проблема та Пітч
**Проблема:** Гравці в більярд часто тренуються хаотично, не фіксуючи результати вправ (наприклад, "свояки" чи "чужі"). Через це важко відстежити реальний прогрес і зрозуміти слабкі сторони.

**Пітч:** Billiard Practice Tracker — це простий інструмент для ведення статистики тренувань. Гравці можуть записувати результати своїх серій ударів, щоб бачити динаміку розвитку навичок, замість того щоб покладатися на відчуття.

## 2. Ядро цінності (MVP)
Мінімально життєздатний продукт (MVP) буде реалізовувати **один ключовий сценарій**:
1. Користувач запускає програму.
2. Вводить назву вправи (наприклад, "Чужий у кут").
3. Вводить результат (кількість забитих куль з 10 спроб).
4. Програма зберігає цей запис у локальний файл з датою тренування.

Без цього сценарію цінність продукту не реалізується.

## 3. Карта Стейкхолдерів
* **Гравець (Користувач):** Головна особа, яка отримує цінність. Хоче витрачати мінімум часу на запис.
* **Тренер:** Може переглядати історію файлу, щоб коригувати план тренувань.
* **Розробник (Student):** Відповідає за реалізацію коду в межах навчального курсу. Обмеження: час на розробку та рівень знань Python.

## 4. Мета та Ознаки успіху
**Мета першої ітерації:** Створити робочий прототип, який дозволяє зробити один повний цикл запису даних без помилок.
