# Практична робота №3: Проєктування інтерфейсу

**Тема:** Дизайн Billiard Practice Tracker

## 1. Щасливий шлях користувача
1. Відкриття дашборду "My stats" (статистика за тиждень).
2. Натискання на золоту кнопку "+" для додавання нового запису.
3. Введення назви вправи та кількості забитих куль.

## 2. Стилістика та ієрархія
- **Кольори:** Тематичний темно-зелений фон (#0B3022) та золотий акцент (#FFD700) для ККД.
- **Шрифт:** Використано Inter Bold для кращого зчитування головного показника (78%).
